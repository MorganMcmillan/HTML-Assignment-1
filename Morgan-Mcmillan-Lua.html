The Lua Programming Language

Introduction

Lua is a dynamically typed embedded scripting language. It has a minimal syntax making it beginner friendly, but contains a powerful C API allowing to be embedded into and used by any application, making Lua capable of modifying the behavior of programs.

History

Lua was created on 1993 in Brazil by a group of computer researchers to use as an embedded data language for customizing software.
Brazil has a strong trade barrier for computer software, which made it impossible to purchase customized software.
The language was influenced by C++, CLU, and Scheme

Current version of Lua, 5.4, was released on June 29th, 2020. It introduced a generational mode for garbage collection, constant variables, and to-be-closed variables.

LuaJit is a Just-In-Time (JIT) compiler for Lua, enabling Lua programs to run faster. It achieves this by compiling Lua code into machine code while it is running, allowing it to take advantage of the CPU's hardware without having to precompile scripts. LuaJit was created by Mike Pall in 2005, and is available under the MIT license.

Usage

Lua was designed to be embedded into applications to enable users to write their own functionality in Lua. This is done using a C API, allowing C code to call Lua code, and Lua code to call C code. This enables users to not only create their own functionality, but to also use functions provided by the application.
